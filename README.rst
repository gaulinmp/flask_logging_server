================================
Flask Boilerplate
================================

Empty Flask app with default User login working.

Quick Start
------------
1) Clone the repo
2) Add pages by:
  a) copying the controllers.user or pages.py from baseapp, then adding new blueprint to loading.
  b) Adding to pages.py in baseapp.controllers.


Run
---------
```sh
$ python run.py
```

Shell
--------
```sh
$ python shell.py
```
